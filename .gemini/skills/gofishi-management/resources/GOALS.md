# Gofishi - Project Goals & Requirements (MVP)

## Business Information
- **Name**: Gofishi (gofishi.com)
- **Niche**: Saltwater Angler & Fly Fishing Specialist.
- **Location**: Jakarta Utara.
- **Partners**: 30+ US Coast Guard licensed captains.

## Level User
1. **Admin**: Tiered (Admin 1, 2, 3) with different authority levels.
2. **Host/Vendor**: Boat owners/captains.
3. **User**: Anglers/Guests.

## Key Features Checklist
- [x] **Tiered Admin**: Admin 1 (Super), Admin 2 (Manager), Admin 3 (Staff).
- [x] **Registration**: 17+ age verification, Email/Phone, Captcha.
- [x] **Boat Listings**: Max 10 photos (max 1MB each), Google Maps integration.
- [x] **Boat Metadata**: KM Name, Captain, Engine Specs, Capacity, Facilities.
- [/] **Alur Sewa (Booking Flow)**: 
    - Daily/Multi-day packages (1, 2, 3 days).
    - Specific Meeting & Return times per package.
    - Specific Area per package.
- [x] **Payments**: Midtrans (QRIS Utama, Bank Transfer).
- [x] **Notifications**: Email & WhatsApp (Booking in, accepted, rejected, paid).
- [x] **Chat**: Real-time chat between User and Host.
- [x] **Reviews**: Rating & reviews with Admin moderation.
- [x] **Reports**: Daily, Weekly, Monthly, and Custom reports.
- [x] **Admin Action**: Change User to Vendor/Host.

## Detailed Boat Attributes (Adjustment)
- Nama KM (Kapal Motor)
- Nama Kapten
- Mesin Kapal 1 & 2
- Kapasitas Penumpang
- Fasilitas Kapal (Sarapan, dll)
- Lokasi (Dermaga & Titik Koordinat)
- Alur Sewa (Harian dengan Jam Kumpul/Kembali & Area Spesifik)
- Status Ketersediaan (Real-time & Host Approval)
