# Changelog - GoFishi Project Development

Daftar perubahan besar selama pengembangan di lingkungan localhost:

## [April 2026] - Modernisasi & Stabilitasi
- **Dashboard Vendor Premium**: Modernisasi UI Dashboard Vendor dengan desain high-fidelity yang terinspirasi dari Airbnb.
- **Wizard Multi-Step**: Implementasi wizard pendaftaran kapal dan lokasi yang intuitif untuk memudahkan vendor non-teknis.
- **Standarisasi Mata Uang**: Penerapan format mata uang Rupiah (IDR) secara konsisten di seluruh dashboard Admin dan Vendor.
- **Optimalisasi Mobile UI**: Perbaikan responsivitas pada header, pencarian, dan elemen navigasi untuk pengalaman mobile yang lebih mulus.
- **Sistem Notifikasi WhatsApp**: Integrasi pengiriman pesan otomatis melalui WhatsApp untuk konfirmasi pesanan dan update status.
- **Pembaruan Footer & Konten Statis**: Penambahan halaman Syarat dan Ketentuan, Kebijakan Pembatalan, dan Kebijakan Kerusakan.
- **Keamanan Otentikasi**: Audit dan perbaikan pada sistem login serta proteksi rute dashboard.
- **Perbaikan Checkout Flow**: Rekonstruksi alur pembayaran dan integrasi backend-frontend untuk meminimalkan error saat transaksi.

## [Sebelumnya] - Fondasi Proyek
- **Integrasi Google Maps**: Fitur penentuan lokasi kapal secara visual pada peta.
- **Sistem Booking**: Alur dasar pemesanan kapal dan pengelolaan ketersediaan.
- **Dashboard Admin**: Panel kendali untuk manajemen user, konten, dan konfigurasi sistem.
- **Localization**: Penerjemahan interface ke Bahasa Indonesia untuk pasar lokal.
