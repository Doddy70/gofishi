Spesialisasi: Fokus pada Marketplace penyewaan perahu dan Saltwater Angler.
* Jasa Pemandu: Bermitra dengan lebih dari 30 kapten berlisensi untuk memandu perjalanan memancing (charter).
* Produk: Menjual berbagai perlengkapan seperti joran, gulungan (reels), pakaian, topi, tas, dan aksesori dari merek-merek ternama.
* Lokasi: Usaha Beroperasi di Jakarta Utara


Basic Requirement : 1. Pembuatan website AirBnB Clone dengan produce penyewaan perahu dan destines wisata

2. Level User: Admin, Host/Vendor, User

3. Fungsi User : cari dan booking, cancel booking, bayar online, beri ulasan dan bintang, lihat riwayat booking

4. Fungsi User Host/Vendor: upload foto maks 10 buah, atur harga dan kalendar, tolak dan terima booking, alamat dan lokasi (google map), fasilitas, jumlah orang, jumlah kamar, info lain.

5. Fungsi User Admin: kelola user dan listing, Approve/Reject/Suspend User Host/Pemilik, monitoring transaksi, moderasi konten, ddan memiliki wewenang merubah user menjadi Vendor/Host

6. User Admin dibagi berjenjang menjadi Admin 1, Admin 2, Admin 3 berjenjang sesuai kebutuhan dan tingkat kewenangan yang diberikan. 
7. Akan ada laporan jadwal, booking, confirmed, canceled, reject, payment, jumlah user, custom sesuai yang diiput oleh user. (daily, weekly, monthly, serta custom).

8. Ada fitur search filter untuk lokasi, harga, tanggal, tamu. 
9. Proses registrasi user diatas 17 tahun, email dan password, no telpon, lupa password, captcha. 
10. Ada notifikasi : booking masuk, booking diterima, booking ditolak, pembayaran berhasil ke dashboard admin atau push notification


Lanjutan :  1. Pengurusan ke Payment Gateway menggunakan metode Qrish (Utama), Transfer Bank, dll (Midtrans)

2. Terdapat chekbox yang menyatakan bawa User harus berusia diatas 17 tahun saat order / booking perahu pada checkout

3 Akan ada notifikasi email atau WhatsApp kepada User, Host dan Admin untuk kondisi:
a. Booking masuk
b. Booking diterima
c. Booking ditolak
d. Pembayaran diterima

4 Ada fitur chat bagi User dan Host untuk setiap postingan Host.

5. Ada fitur untuk pemberian rating dan ulasan. User, Host dapat memberikan ulasan dan tanggapan atas chat yang disampaikan. User Admin berwenang melakukan penghapusan atas ulasan yang tidak sesuai dengan etika dan peraturan website. 

6. Fitur untuk USER GUEST: 
a. Search by Tanggal
b. Search by Lokasi
c. Search by jumlah Tamu
d. Memberikan Ulasan dan Bintang setelah selesai transaksi terjadi
e. Chat ke Host
f. Melihat Postingan dan Ulasan Host
g. User Guest tidak perlu login dulu untuk melihat lihat postingan, login saat akan melakukan pembayaran

a. Upload Foto maksimal 10 foto (pembatasan besar file per foto maks 1 Mb) b. Atur tanggal ketersediaan
c. Atur harga sewa

Mengisi TEMPLATE Deskripsi Objek yaitu: Ukuran Kapal (Panjang x Lebar), Fasilitas, Kapasitas/Jumlah orang, Waktu berangkat, Waktu tiba, Jumlah awak kapal, Umpan, Peralatan Pancing, Baju Pelampung, Makan pagi, Makan siang, Makan malam, Air Minum, Mesin Kapal 1, Mesin Kapal 2, AC, Kamar Tidur, Toilet, Wifi, Listrik, Kompor, Kulkas, Lain-lain.
