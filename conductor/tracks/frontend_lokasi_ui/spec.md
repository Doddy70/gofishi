# Specification: frontend_lokasi_ui

## Summary
Transform the existing Location/Hotel Details page into a premium "Jay Peak Resort" Airbnb-style destination page, and resolve related vendor/admin backend issues.

## Acceptance Criteria
- [x] Full-width hero image with a floating search card inside it.
- [ ] A functional horizontal carousel instead of stacked grids for available boats.
- [x] Quick Stats, Verified Amenities, FAQ Sections correctly pulling data.
- [x] Admin can see all vendors without being restricted by subscription.
- [x] Existing Early-Adopter Vendors have unlimited/lifetime memberships.
- [ ] Vendor can successfully add boats linked to these locations.
